# 📊 Emp Manager (Менеджер выплат сотрудникам)

Современное WPF-приложение (C# / .NET) для обработки, агрегации и управления данными о выплатах сотрудникам на основе XML и XSLT-преобразований. 

Разработано в рамках тестового задания. Особенность проекта — полностью кастомный адаптивный пользовательский интерфейс в темной теме (Dark Mode), спроектированный без использования сторонних UI-библиотек (только чистый XAML).

## ✨ Ключевые возможности

### ⚙️ Работа с данными (Логика)
* **Загрузка XML:** Чтение исходных файлов с данными о выплатах (`Data1.xml` / `Data2.xml`).
* **XSLT-трансформация:** Преобразование исходного XML в целевой формат с помощью `Transform.xslt`.
* **Агрегация (LINQ to XML):** 
  * Автоматический подсчет итоговой суммы выплат по каждому сотруднику.
  * Группировка и подсчет общих сумм выплат по месяцам с правильной хронологической сортировкой.
* **Добавление записей:** Возможность добавить новую выплату сотруднику прямо из интерфейса с моментальным пересчетом всех таблиц.
* **Экспорт:** Сохранение итогового сформированного файла `Employees.xml` в выбранную пользователем директорию.

### 🎨 Пользовательский интерфейс (UI/UX)
* **Modern Dark Theme:** Глубокая темная палитра с неоновыми акцентами (фиолетовый и изумрудный).
* **Адаптивность (Responsive):** Использование `GridSplitter` и пропорциональной верстки позволяет пользователю свободно менять размеры панелей и самого окна без "поломки" дизайна.
* **Custom Controls:** Полностью переписанные шаблоны (ControlTemplate) для стандартных контролов WPF:
  * Объемные интерактивные кнопки с анимацией нажатия.
  * Стилизованный `ComboBox` со скругленным выпадающим меню.
  * Тонкий современный `ScrollBar`.
  * Переработанный `DataGrid` без стандартных системных рамок с подсветкой строк.
* **Custom MessageBox:** Системные окна заменены на собственные стилизованные всплывающие уведомления, поддерживающие перетаскивание (DragMove) и закрытие по клавише `Enter`.

## 🛠 Технологии
* **Платформа:** WPF, .NET
* **Язык:** C#
* **Работа с данными:** `System.Xml.Linq` (XDocument), `System.Xml.Xsl` (XslCompiledTransform)
* **UI:** XAML (без использования MaterialDesignThemes или MahApps, все стили написаны с нуля).

## 🚀 Как запустить и использовать

1. Склонируйте репозиторий и откройте решение в Visual Studio (или Rider).
2. Соберите проект.
3. **В приложении:**
   * Нажмите **«Загрузить XML-файл»** и выберите исходный файл с данными.
   * Просматривайте итоговые суммы в таблицах справа.
   * Для добавления выплаты заполните форму слева и нажмите **«Добавить и пересчитать»**.
   * По завершении работы нажмите **«Сохранить XML-файл»**, чтобы выгрузить результат.

## 📂 Структура данных

**Пример исходного файла (`Data1.xml`):**
```xml
<items>
  <item name="John" surname="Doe" amount="1500.50" mount="january" />
  <!-- ... -->
</items>
```

**Пример итогового файла (`Employees.xml`):**
```xml
<Employees>
  <Employee name="John" surname="Doe" totalSalary="1500.50">
    <salary mount="january" amount="1500.50" />
    <!-- ... -->
  </Employee>
</Employees>
```
